# Техническая структура и Стек

## Стек технологий
- **Язык**: TypeScript (на обеих сторонах).
- **Backend**: Node.js, Fastify или Express.
- **Бот**: GrammY или Telegraf.
- **Database**: SQLite (с Prisma ORM) для надежности и простоты.
- **Frontend**: Vite + React.
- **Styling**: Vanilla CSS + CSS Variables (для Glassmorphism).
- **State Management**: Zustand.

## Модель данных (Основные сущности)
- **User**: id, telegram_id, role (parent/child), name, points.
- **Task**: id, title, description, reward, status (active/pending/completed), assignee_id (null for global).
- **StoreItem**: id, title, price, description, icon.
- **Purchase**: id, user_id, item_id, status.

## CLI Команды (Пульт управления)
- `users add parent <tg_id> <name>`
- `users add child <tg_id> <name>`
- `users list`
- `stats show`
